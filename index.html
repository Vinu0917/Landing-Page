The Problem

Traditional Hela Bojunhala operations rely heavily on manual processes, resulting in slow service, poor coordination, lack of transparency, and limited digital support.

  • Manual order taking leads to inefficiency and errors.
  • Limited visibility into stock and order status across branches.
  • Slow communication between cashiers, drivers, and admins.
  • Difficulty scaling operations to multiple locations.

The Solution

Our system digitizes these processes to improve speed, accuracy, transparency, and customer satisfaction.

Hela Bojunhala – Smart Delivery & Business Management System

We are planning and developing a mobile application using React Native to bring authentic Hela Bojun foods right to your doorstep.

Our system modernizes Hela Bojunhala operations by automating order management, payments, and delivery coordination across multiple branches.

Connecting Customers, Cashiers, Drivers, and Admins seamlessly.

Customer Portal

Users can browse menus, place online orders, make secure payments, and track their deliveries in real-time.

Cashier Portal

Cashiers manage order confirmations, monitor kitchen status, and coordinate with drivers seamlessly.

Driver Portal

Drivers receive delivery tasks, update progress, and complete orders efficiently with real-time instructions.

Admin Portal

Admins oversee all operations, manage vendors, review reports, and analyze business insights.

Main Portals

Customer Portal (React Native)

  • Browse menu filtered by ingredients, health preferences, availability.
  • Place orders directly to a selected branch.
  • Track order status in real-time.

Cashier Portal

  • Manage incoming orders from all branches.
  • Handle payments, generate receipts, update order status.
  • Communicate with drivers efficiently.

Driver Portal

  • Receive automatic order assignments.
  • Track delivery routes and update delivery status.
  • View branch-wise pending and completed deliveries.

Admin Portal

  • Analyze sales, orders, and driver performance across branches.
  • Generate reports for branch management.
  • Monitor system health and ensure smooth operations.

Architecture / System Design

Frontend

React Native mobile app for all portals.

Backend

Node.js with Express.js providing RESTful APIs.

Database

MySQL storing users, orders, branches, and analytics.

Flow

Customer → Cashier → Driver → Admin → Analytics.

Technologies / Tools

Frontend

  • React Native – Cross-platform mobile app development.
  • React Navigation – Smooth multi-screen navigation (stack, tabs, drawer).
  • Axios – HTTP client with interceptors for JWT auth.

Backend

  • Node.js / Express.js – RESTful API server.
  • JWT – Secure, stateless authentication and role-based access control.
  • Sequelize / Knex – ORM/query builder for MySQL with migrations.

Database

  • MySQL – Persistent storage for orders, users, branch details.

Tools

  • VS Code, Postman, GitHub.
